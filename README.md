# Superstore SQL Data Analysis

## Overview
This project performs end-to-end data analysis on a Superstore sales dataset
using MySQL. The data was imported from CSV and analyzed to extract
business insights.

## Tools Used
- MySQL
- MySQL Workbench

## Dataset
- Superstore sales data
- Imported from CSV file

## Work Done
- Cleaned and renamed columns
- Converted date formats
- Created derived features (year, month, weekday)
- Classified orders as weekday or weekend
- Performed sales and profit analysis

## SQL Concepts Used
- Aggregate functions (SUM, AVG, COUNT)
- CASE and IFNULL
- GROUP BY and ORDER BY
- Window functions (RANK)
- Date functions

## Files
- schema.sql
- data_cleaning.sql
- analysis_queries.sql
